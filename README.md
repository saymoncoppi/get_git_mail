# Get Git Mail
It's a pretty bash script to get user email account from GitHub page. 
Howto:
- Download the ggm.sh
- Make it executable (`sudo chmod +x ggm.sh`)
- Later Run at terminal
~~~
Run ./ggm.sh UserName
~~~
